## Full description (product page)

**Stop talking to a chat box. Take the command chair.**

DATA is a self-hosted dashboard that wraps the AI you already use in a mission-control interface — built for people who want to *work* with AI, not just message it.

Point it at your own Claude, ChatGPT, Gemini, or a free local Ollama model. DATA does the rest: a main channel for real work, a crew of ten specialist agents, memory that persists across sessions, parallel project workspaces, scheduled "standing orders," and a live readout of your machine — all wrapped in a dark, elegant, slightly-futuristic cockpit you'll actually want to look at.

**What you get**
- **A crew, not a chatbot** — Atlas (strategy), Forge (building), Vector (review), Sentinel (security), Probe (debugging), Relay (ops), and more. Switch the agent driving the chat with one click.
- **Persistent memory** — DATA remembers across sessions, with a searchable archive of every conversation.
- **Project workspaces** — spawn parallel chat panes rooted in any folder, each with its own AI provider and role. Run Claude and Codex side by side.
- **Standing orders** — schedule AI tasks to run on their own (daily reports, checks, scans).
- **Potential Upgrades** — DATA scans for new AI tools, MCP servers, and skills you might want.
- **Live system vitals** — real CPU/RAM/GPU readouts in a full Master Systems Display.
- **Two dark themes** — a clean, professional MINIMAL and a neon CYBER, one click apart.

**Use the AI you already pay for**
DATA needs no API keys from us. It drives the official command-line tool for whichever provider you choose and logs in with *your* account:
- Claude (Pro/Max subscription) — recommended
- GPT-5 Codex (ChatGPT subscription)
- Gemini (Google account)
- Ollama (free, fully local, no account)

**Yours, private, forever**
Everything runs on your computer. No telemetry, no cloud, no accounts with us. Your memory, history, and files never leave your machine. One-time purchase — no subscription.

**Runs on** Windows, macOS, Linux, and Chromebooks (Linux container). Requires Python 3.10+ (free) and one of the AI tools above.
