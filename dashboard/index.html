Crew voice

Which officer's voice is used for spoken replies (the ▶ play button on Data's messages).

Speech-to-text model

What turns your speech into text. Smaller = much faster on a CPU / no-GPU machine (e.g. a Surface). Larger = more accurate. ⤓ = not downloaded yet; picking it installs it. Applies on your next spoken turn.

Recognition speed vs accuracy

Beam size. 1 = greedy = fastest (recommended for slow machines). Higher decodes several ways and ranks them — more accurate, more CPU, slower.

Compute precision

int8 is fastest on a CPU and is the default. The float types only help if you have a GPU.

Default LLM
